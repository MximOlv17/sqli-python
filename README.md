# FEYR Project
## **METU SCHOLARSHIP GOAL**

FEYR is a SQLi injection script that automates the process of a BLIND SQLi
FEYR has 2 main funtions
1) Search
2) Length

-Search
  Search function is made to get the char dependind on the SQLi query you input

-Length
  Length function is made to get the length of any db item

### FEYR Core Functions
* **`length_search`**: Calculates string length to define the extraction boundaries.
* **`char_search`**: Implements **Binary Search** logic to minimize server overhead and maximize speed.
* **Threading**: Utilizing `concurrent.futures` for high-speed parallel data retrieval.
* **Proxy Support**: Integrated proxy-handling for anonymized security assessments.
